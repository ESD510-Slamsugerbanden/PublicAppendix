# Initial search test results
To compensate for the drift found in the encoders, a very low frequency high pass filter is utilized to compensate for some of this.

The following files can be found:

**POSX_16vinkel.csv**: The spacial sweeps for a given source location see report for the map.
**Stationary accuracy mechanial.py** The code for generating the histogram given a set of distance measurements
**Initial_Tracking_stacked.py** Plots for the spacial sweeps and where the source were estimated to be.
